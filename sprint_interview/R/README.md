# Q. Паша и Марк

## Формат ввода

На вход подаётся единственное число n (1 ≤ n ≤ 10000).

## Формат вывода

Выведите «Pasha», если выиграет Паша, или «Mark», если победа за Марком.

### Пример 1

<td>
8
</td>
<td>
Pasha
</td>
</tr></table>
